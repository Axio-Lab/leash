## Prerequisites

- Node **>= 20**
- pnpm **9**
- (Optional) Docker for the compose flows

## Install + build

```bash
pnpm install
pnpm turbo run build typecheck lint test
```

## Run the split stack (default)

```bash
docker compose up --build
```

Brings up `runner` (`:8787`), `seller-demo` (`:3001`), and `web` (`:3000`).

## Run the merged stack

```bash
docker compose --profile merged up --build merged-demo
```

A single agent that both sells (`POST /echo`) and buys (interval-driven calls back into itself), all writing to one receipt feed.

## Scripted demo

```bash
SELLER_URL=http://localhost:3001 pnpm exec tsx scripts/e2e-demo.ts
```

## Hosted playground at a glance

Once the web app is up at [http://localhost:3000](http://localhost:3000):

| Surface                 | What it is                                                                                                    | Signing model                                                                |
| ----------------------- | ------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| `/agents/new`           | Mint a Core asset + Agent Identity. Set behaviour rules (or "limitless") and the initial USDC spend allowance. | Privy embedded wallet (asset owner). Approves the executive as SPL delegate. |
| `/agents/[mint]`        | Inspect identity, treasury balances, and the live spend allowance. Top up, adjust, or revoke the delegation.   | Privy embedded wallet.                                                       |
| `/seller`               | **Payment-Link Builder.** Pick an agent → declare price + response → get a shareable `/x/<id>` x402 paywall.  | Runner stores descriptors; `/x/<id>` settles via `facilitator.svmacc.tech`.  |
| `/buyer`                | **Autonomous-agent cockpit.** Pick an agent → fire any x402 URL with that agent's stored behaviour rules.     | Privy wallet signs as the agent's registered Executive (no keys in browser). |

See [Create a payment link](/guides/create-an-endpoint),
[Fund an agent](/guides/fund-an-agent), and
[Build a buyer](/guides/build-a-buyer) for end-to-end walkthroughs.
