## Pick your surface

If you just want to **drop Leash into an AI agent** (Cursor, Claude
Desktop, Cline, …), you do not need to clone this repo at all. Skip
straight to one of:

- [Standalone MCP server](/agents/mcp) — `npx -y @leash/mcp` and the
  agent can mint, pay, and discover on Solana from the chat.
- [Human CLI — `leash`](/agents/cli) — `npm i -g @leash/cli` for
  terminal ops on the same identity.
- [TypeScript SDK — `@leash/sdk`](/agents/sdk) — typed
  `LeashClient` for app developers.

The rest of this page is for running the **whole Leash stack
locally** (runner, seller, buyer, web playground, indexer) — useful
if you're working on the protocol itself or building a custom
deployment.

## Prerequisites

- Node **>= 20**
- pnpm **9**
- (Optional) Docker for the compose flows

## Install + build

```bash
pnpm install
pnpm turbo run build typecheck lint test
```

## Run the split stack (default)

```bash
docker compose up --build
```

Brings up `runner` (`:8787`), `seller-demo` (`:3001`), and `web` (`:3000`).

## Run the merged stack

```bash
docker compose --profile merged up --build merged-demo
```

A single agent that both sells (`POST /echo`) and buys (interval-driven calls back into itself), all writing to one receipt feed.

## Scripted demo

```bash
SELLER_URL=http://localhost:3001 pnpm exec tsx scripts/e2e-demo.ts
```

## Hosted playground at a glance

Once the web app is up at [http://localhost:3000](http://localhost:3000):

| Surface          | What it is                                                                                                                                                                                                   | Signing model                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------- |
| `/agents/new`    | Mint a Core asset + Agent Identity: paste your **hosted Core metadata JSON URL**, configure services/trust/rules, then mint. Spend delegation is **not** configured here — use the agent profile after mint. | Privy embedded wallet (asset owner).                                                                |
| `/agents/[mint]` | Identity, treasury, execute (executive + delegation), optional Genesis **token** launch, and spend delegation / funding controls.                                                                            | Privy embedded wallet.                                                                              |
| `/seller`        | **Payment-Link Builder.** Settlement currency + optional "also accept" stables → shareable `/x/<id>` x402 paywall.                                                                                           | Runner stores descriptors; `/x/<id>` settles via `devnet-facilitator.leash.market` (Leash default). |
| `/buyer`         | **Autonomous-agent cockpit.** **Pay with** (USDC / USDT / USDG) + policy rules → fire x402 URLs; treasury readout follows the selected mint.                                                                 | Privy wallet signs as the agent's registered Executive (no keys in browser).                        |

See [Create a payment link](/guides/create-an-endpoint),
[Fund an agent](/guides/fund-an-agent), and
[Build a buyer](/guides/build-a-buyer) for end-to-end walkthroughs.
