Leash ships a wallet (an MPL Core agent treasury PDA), a capped spend allowance
(SPL token delegate to your executive wallet), a policy gate (`RulesV1`),
and a chained receipt feed (`ReceiptV1`) — all settled through real
[x402](https://x402.org) HTTP payments in SPL stables (USDC, USDT, USDG) on
Solana, via any compatible facilitator. Drop it in front of any agent and it
can buy, get paid, and prove every transfer happened.

## The five surfaces

Leash gives an agent the same identity, treasury, and reputation no
matter where it runs. Pick the surface that matches your host:

| Surface                                 | Lives at                                                 | What it's for                                                                                                                                                                                                                                                                                                 |
| --------------------------------------- | -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Standalone MCP server**               | `npx -y @leashmarket/mcp` or `leash-mcp`                 | Drop into Cursor / Claude Desktop / Cline / Continue / ChatGPT-MCP. STDIO transport, 14 tools. ([Agents → MCP](/agents/mcp))                                                                                                                                                                                  |
| **Human CLI** — `leash`                 | `npm install -g @leashmarket/cli`                        | Mint, inspect, and operate agents from the terminal. Same `agent.json` as the MCP. ([Agents → CLI](/agents/cli))                                                                                                                                                                                              |
| **TypeScript SDK** — `@leashmarket/sdk` | npm                                                      | Typed `LeashClient` for app developers. Discover services, vet reputation, manage webhooks. ([Agents → SDK](/agents/sdk))                                                                                                                                                                                     |
| **Agent platform**                      | [`agent.leash.market`](https://agent.leash.market)       | Hosted chat product. Mint an agent and run it from the browser; share the same on-chain identity with MCP/CLI.                                                                                                                                                                                                |
| **MCP marketplace**                     | [`leash.market`](https://leash.market)                   | Open registry of paid services. Discoverable from agents via `leash_discover` / `leash discover` / `leash.discover()`. Discovery also surfaces the [Solana Foundation `pay-skills`](https://github.com/solana-foundation/pay-skills) registry alongside Leash listings, tagged via a per-item `source` field. |
| **API** — REST + OpenAPI                | [`api.leash.market`](https://api.leash.market)           | Polyglot HTTP surface. Prepare/submit Solana txs, ingest receipts, monetise APIs. Spec at `/openapi.json`.                                                                                                                                                                                                    |
| **Explorer** — protocol UI              | [`explorer.leash.market`](https://explorer.leash.market) | Solscan-style search across agents, transactions, receipts, and events. Dual-network.                                                                                                                                                                                                                         |

Pick whichever fits the layer you're building. They all read and write
the same on-chain state, the same receipt schema, and the same event
model.

## Open-source SDK

- **Packages** — `@leashmarket/schemas`, `@leashmarket/core`, `@leashmarket/registry-utils`, `@leashmarket/seller-kit`, `@leashmarket/buyer-kit`, `@leashmarket/runner`, `@leashmarket/testing`
- **Apps** — `web` (playground), `docs` (this site), `seller-demo`, `buyer-demo`, `merged-demo`, `agent-token-demo`
- **Scripts** — `fund-devnet`, `upload-registration` (fetch + validate `RegistrationV1` from a URL), `e2e-demo`, `gen-schema-docs`

The hosted [API](/api/overview) and [Explorer](/explorer/overview)
are operated by Leash and consume the same on-chain primitives
these packages produce.

## For LLMs and coding agents

These docs are designed to be slurped wholesale into your model's context.

| Resource                                                                 | What it gives you                                                                                                                                      |
| ------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [`/llms.txt`](https://docs.leash.market/llms.txt)                        | A curated, sidebar-grouped index. Every entry links to the Markdown rendering of the page. Auto-discovered via the `Link: rel="llms-txt"` HTTP header. |
| [`/llms-full.txt`](https://docs.leash.market/llms-full.txt)              | The entire site concatenated into one Markdown file. Drop into a system prompt and you have the whole manual.                                          |
| `/<page>.md`                                                             | Append `.md` to **any** docs URL to fetch the raw Markdown for a single page (e.g. [`/api/overview.md`](https://docs.leash.market/api/overview.md)).   |
| [`api.leash.market/openapi.json`](https://api.leash.market/openapi.json) | Live OpenAPI 3.1 spec. Feed it to your client generator of choice.                                                                                     |
| **In-page actions**                                                      | The header on every page has a "Copy page" / "View as Markdown" / "Open in ChatGPT, Claude, Perplexity" / "Connect to Cursor or VS Code (MCP)" menu.   |

Building on top of Leash from inside an agent (Cursor, Claude Code, Codex,
Replit, Windsurf)? Install the **Leash skill** so your assistant always has
the right primitives at hand:

```bash
git clone https://github.com/leash-market/leash ~/leash
mkdir -p .cursor/skills && ln -sf ~/leash/skills/leash .cursor/skills/leash
# (or: .claude/skills, .codex/skills, .windsurf/skills — see skills/leash/INSTALL.md)
```

The skill ships a curated `SKILL.md` (mental model + workflow), a
`REFERENCE.md` (full surface map — every SDK package, every API route,
every env var), and an `EXAMPLES.md` of copy-paste snippets for the
most common flows. Source: [`leash/skills/leash`](https://github.com/leash-market/leash/tree/main/skills/leash).
