**Leash is stablecoin rails for agents that spend on the open internet.** It
ships a wallet (an MPL Core agent treasury PDA), a capped spend allowance
(SPL token delegate to your executive wallet), a policy gate (`RulesV1`),
and a chained receipt feed (`ReceiptV1`) — all settled through real
[x402](https://x402.org) HTTP payments in USDC on Solana, via any compatible
facilitator. Drop it in front of any agent and it can buy, get paid, and
prove every transfer happened.

## Monorepo

- **Packages** — `@leash/schemas`, `@leash/core`, `@leash/registry-utils`, `@leash/seller-kit`, `@leash/buyer-kit`, `@leash/runner`, `@leash/testing`
- **Apps** — `web` (explorer), `docs` (this site), `seller-demo`, `buyer-demo`, `merged-demo`
- **Scripts** — `fund-devnet`, `upload-registration`, `e2e-demo`, `gen-schema-docs`

See the repo `README.md` for the 5-minute demo and environment variables.
