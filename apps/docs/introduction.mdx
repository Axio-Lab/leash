Leash is a **wallet + policy + receipt** stack for autonomous agents on Solana.

## Monorepo

- **Packages** — `@leash/schemas`, `@leash/core`, `@leash/registry-utils`, `@leash/seller-kit`, `@leash/buyer-kit`, `@leash/runner`, `@leash/testing`
- **Apps** — `web` (explorer), `docs` (this site), `seller-demo`, `buyer-demo`, `merged-demo`
- **Scripts** — `fund-devnet`, `upload-registration`, `e2e-demo`, `gen-schema-docs`

See the repo `README.md` for the 5-minute demo and environment variables.
