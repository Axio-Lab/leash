# @leash/core

Policy evaluation, receipt hashing / chain verification, x402-shaped `x402Fetch`, treasury helpers, and env kill-switch (`readPauseFromEnv`).

## Usage

```ts
import { evaluate, finalizeReceipt, verifyReceiptChain, x402Fetch } from '@leash/core';
```

## Test

```bash
pnpm --filter @leash/core test
```
