# @leash/schemas

Zod source of truth for **ReceiptV1**, **RulesV1**, **LeashBlockV1**, **RegistrationV1**, plus JSON Schema exports under `dist/schemas/` after `pnpm build`.

## Usage

```ts
import { ReceiptV1Schema } from '@leash/schemas';
```

## Build

```bash
pnpm --filter @leash/schemas build
```
